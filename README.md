## üöÄ**Quick Start**
1. **Clone project**
    ```sh
    $ git clone https://github.com/sky172839465/leetcode-in-js.git
    ```
2. **Install dependency**
    ```sh
    $ npm install
    ```
3. **Run single solution**
    ```sh
    $ npm run test __tests__/easy/001-two-sum.test.js
    ```
    **Run all solutions**
    ```sh
    $ npm run test
    ```
4.  **Solution create / remove**
    ```sh
    $ npm run solution
    ```
